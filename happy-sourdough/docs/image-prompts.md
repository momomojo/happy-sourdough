## Notes for Image Generation

1. **Consistency**: Maintain warm cream/golden color palette across all images
2. **Quality**: Request high resolution (at least 2048px) for hero images
3. **Variations**: Generate 3-4 variations of each product shot and select best
4. **Background**: Prefer simple, slightly blurred backgrounds to focus on product
5. **Props**: Use consistent props - linen napkins, wooden boards, vintage ceramics
6. **Lighting**: Soft natural light, avoid harsh shadows
7. **Text space**: Leave room for overlays in marketing images
