## Image Mapping File

**Location**: `/src/data/image-mapping.ts`

This TypeScript file exports:

### Constants
- `LOGO` - Logo image paths
- `HERO` - Hero section images
- `PRODUCTS` - Product images by category (breads, pastries, cakes, cookies)
- `CATEGORIES` - Category header images
- `LIFESTYLE` - Lifestyle/scene images (delivery, kitchen, cafe, picnic)
- `UI` - UI state images (empty cart, order confirmed, baking in progress)
- `SOCIAL` - Social media images (Instagram, Facebook)

### Helper Functions
- `getProductImage(category, productName)` - Get product image by category and name
- `getCategoryHeader(category)` - Get category header image
- `getRandomUIVariant(uiType)` - Get random UI variant for state images

### Type Exports
- `LogoKey`, `HeroKey`, `CategoryKey`
- `ProductCategory`, `BreadProduct`, `PastryProduct`

### Usage Example
```typescript
import { LOGO, PRODUCTS, HERO } from '@/data/image-mapping'

<Image src={LOGO.PRIMARY} alt="Happy Sourdough" />
<Image src={PRODUCTS.BREADS.CLASSIC_SOURDOUGH} alt="Classic Sourdough" />
<Image src={HERO.HOMEPAGE} alt="Bakery" />
```
