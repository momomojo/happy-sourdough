  3. Context Optimization Strategies

  | Strategy           | Reduction | Implementation               |
  |--------------------|-----------|------------------------------|
  | Tool Search Tool   | 85%       | defer_loading: true on tools |
  | Lazy-MCP Proxy     | 95%       | On-demand tool activation    |
  | Tool Consolidation | 60%       | Reduce 20â†’8 tools            |
  | JSON Trimming      | 93-98%    | Minimal API responses        |
  | McPick CLI         | Variable  | Toggle servers per session   |

  The Problem: MCP tools can consume 66,000-134,000 tokens before conversation starts.

  Solution Stack:
  1. Use Docker MCP Toolkit for unified management
  2. Enable only needed servers per task
  3. Implement lazy loading patterns
  4. Use MAX_MCP_OUTPUT_TOKENS env var (default 25,000)

  Phase 3: Custom Agents to Create

  1. sourdough-content-gen Agent

  Purpose: Generate sourdough-specific content
  Triggers: "create recipe", "write product description", "generate blog post"
  Tools: WebSearch, Write, Read
  Specialization:
  - Sourdough terminology
  - Recipe formatting
  - SEO for bakery businesses

  2. e2e-test-runner Agent

  Purpose: Automated visual and functional testing
  Triggers: After feature completion, before PR merge
  Tools: Playwright MCP, browser_snapshot, browser_click
  Specialization:
  - Mobile responsive testing
  - Checkout flow validation
  - Form submission testing

  3. deploy-manager Agent

  Purpose: Manage deployments and environments
  Triggers: "deploy to staging", "release to production"
  Tools: Bash, GitHub MCP
  Specialization:
  - Vercel/Netlify deployments
  - Environment variable management
  - Rollback procedures

  Phase 4: Custom Skills to Create

  1. sourdough-branding Skill

  Purpose: Ensure consistent brand voice and visual identity
  Includes:
  - Color palette definitions
  - Typography guidelines
  - Tone of voice examples
  - Logo usage rules

  2. recipe-formatter Skill

  Purpose: Format recipes consistently
  Includes:
  - Ingredient list structure
  - Step-by-step instructions format
  - Timing and yield calculations
  - Nutritional info templates

  3. accessibility-checker Skill

  Purpose: Ensure WCAG compliance
  Includes:
  - Color contrast requirements
  - Alt text guidelines
  - Keyboard navigation patterns
  - Screen reader compatibility
